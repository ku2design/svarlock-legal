# SvarLock Legal Pages

Public Privacy Policy and Terms of Service for the [SvarLock](https://github.com/ku2design/svarlock) Android app, hosted via GitHub Pages so they satisfy Google Play's public-URL requirement without exposing the app's private source repo.

Live at: https://ku2design.github.io/svarlock-legal/

Source of truth for the legal text lives in `PRIVACY_POLICY.md` / `TERMS_OF_SERVICE.md` in the main app repo. When those change, copy the updates into `privacy-policy.html` / `terms-of-service.html` here.
